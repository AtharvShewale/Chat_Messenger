# Chat Messenger (Java)

A **server-client console chat application** in Java using **socket programming**.

---

## Features
- Real-time messaging between server and client
- Simple console interface
- Platform-independent

---

## How to Run
1. Open terminal in project folder.
2. Compile files:

javac Server.java
javac Client.java

> **Important:** Run the **Server first**, then the **Client**. Both use the same `localhost` address.

### Step 1: Open two separate terminals
- **Terminal 1** → for **Server**  
- **Terminal 2** → for **Client**

### Step 2: Compile the Java files
In both terminals (or just once), navigate to the project folder and compile:

```bash
cd E:\Projects\Chat_Messenger
javac Server.java
javac Client.java
